ตั้งค่าการเชื่อมต่อครั้งแรก ⚙️

เนื่องจากเปิดใช้งานนอกระบบพรีวิว กรุณาเลือกโหมดที่เหมาะสมกับคุณครูครับ

ทดสอบด่วน

1. โหมดทดลอง (Demo)

เล่นจำลองบนเครื่องคอมพิวเตอร์หรืออุปกรณ์เครื่องเดียวกันได้ทันที **โดยการเปิดแท็บใหม่แบ่งครึ่งจอ** เพื่อทดสอบระบบได้โดยไม่ต้องตั้งค่าฐานข้อมูลคลาวด์

เล่นจริงในห้องเรียน

2. โหมดคลาวด์ออนไลน์

เชื่อมต่อกับ **Firebase Firestore** เพื่อให้นักเรียนสามารถสแกน QR Code แล้วใช้สมาร์ตโฟนของตัวเองร่วมกดกระดิ่งแย่งสิทธิ์จากทั่วห้องเรียนได้ฟรีแบบเรียลไทม์

💡 วิธีรับรหัสเชื่อมต่อฟรีง่าย ๆ:

  1. ไปที่เว็บ Firebase Console แล้วกดสร้างโปรเจกต์ใหม่
  2. กดไอคอน Web </> เพื่อสร้าง App ในระบบ
  3. คัดลอกข้อความในปีกกา firebaseConfig = { ... } นำมาวางในกล่องข้อความนี้
  4. เปิดใช้งาน **Firestore Database** และ **Anonymous Sign-In** บนแถบเมนูข้างใน Firebase Console ด้วยครับ

ยินดีต้อนรับสู่เกมตอบคำถาม!
